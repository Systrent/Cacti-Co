## **âŠ™ Instalation**

---

Clone the project with SSH or HTTPS
```
git clone git@github.com:Systrent/Cacti-Co.git
```

Install dependencies
```
yarn install
```

Start the application on each one, frontend and backend
```
yarn run dev
```
